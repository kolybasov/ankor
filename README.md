# Ankor

## Requirements

* python 3
* pip 3
* sqlite 3

## Installation

* `$ git clone https://github.com/kolybasov/ankor && cd ankor`
* `$ make init`
* `$ [sudo] chmod +x ./run.py`

## Usage

* Copy URL with length 30+ symbols and open url from config.
Default is [http://127.0.0.1:3333](http://127.0.0.1:3333)

## Tests

* `$ make test`
